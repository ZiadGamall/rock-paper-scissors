if 0 then win

game function
make a counter for computer
make a counter for player
make a loop of five counts
if player wins
    increase player counter
if computer wins
    increase computer counter
after five rounds:

if player counter > computer counter
    declare the player to be the winner

if computer counter > player counter
    declare computer to be the winner

NOTE: if a round is a draw decrease the round counter by 1
